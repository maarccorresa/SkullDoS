Herramienta de pruebas de estrés y análisis de resistencia de red.

**USO RESPONSABLE ÚNICAMENTE**

## 📋 Descripción

SkullDoS es una herramienta de testing de red escrita en Python que permite realizar pruebas de estrés controladas para evaluar la resistencia de sistemas y redes. Está diseñada para profesionales de seguridad y administradores de sistemas que necesitan validar la capacidad de sus infraestructuras.

## ✨ Características

- Múltiples modos de testing
- Configuración de parámetros personalizables
- Interfaz de línea de comandos intuitiva
- Reportes de resultados en tiempo real
- Control de intensidad y duración

## 🔧 Requisitos

- Python 3.x
- Dependencias listadas en `requirements.txt`
- Windows (para setup.bat) o Linux/Mac

## 📦 Instalación

### Linux/Mac:

```bash
git clone https://github.com/c0rresita/SkullDoS.git
cd SkullDoS
pip install -r requirements.txt
```

### Windows:

```bash
git clone https://github.com/c0rresita/SkullDoS.git
cd SkullDoS
setup.bat
```

## 🚀 Uso

```bash
python3 skullDoS.py
```

### Parámetros Principales

- **Target**: Dirección IP o dominio objetivo
- **Port**: Puerto a probar
- **Threads**: Número de hilos concurrentes
- **Duration**: Duración de la prueba
